官方 解答
牌局开始每个人抓五张牌，玩家需要将手中三张牌10点的倍数，称为“牛”。其余的两张牌加起来算点数个位数是几就称为牛几，牛的大的赢。如果个位数为0，则称为牛牛，是最大的。如果找不到三张能凑成整10的，则没有牛，算失败了。

举个例子，拿到了8，7，5，4，2，其中8，7，5可以凑成整10，4和2相加得6，则留6。再有，如果拿到了8，7，5，10，J，其中8，7，5可以凑成整10，10和J相加后个位数为0，所以牛牛。再有，如果拿到了6，7，4，1，8，由于找不到任意三张能春兰凑成整10的，所以没有牛，失败了。
